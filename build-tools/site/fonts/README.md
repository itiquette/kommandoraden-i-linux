# Font Assets

These files are used by the published editions.

- `source-serif-4-vf.ttf`, `source-serif-4-italic-vf.ttf`: Source Serif 4 (`SIL OFL 1.1`). HTML body font.
- `atkinson-hyperlegible-*.ttf`: Atkinson Hyperlegible (`SIL OFL 1.1`). HTML headings and UI; PDF body and headings.
- `inconsolata-*.ttf`: Inconsolata (`SIL OFL 1.1`). HTML and PDF monospace.

The corresponding license texts are stored in this directory as `LICENSE-*.txt`. SIL OFL 1.1 permits redistribution of these binaries together with their license texts, which is why they ship in this repository.
