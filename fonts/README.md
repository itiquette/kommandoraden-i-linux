# HTML Font Assets

These files are used by the static HTML reading edition.

- `source-serif-4-vf.ttf` and `source-serif-4-italic-vf.ttf`: Source Serif 4 (`SIL OFL 1.1`)
- `jost-*.ttf`: Jost (`SIL OFL 1.1`)
- `inconsolata-*.ttf`: Inconsolata (`SIL OFL 1.1`)

The corresponding license texts are stored in this directory as `LICENSE-*.txt`.
