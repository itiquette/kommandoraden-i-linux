# Kommandoraden i Linux

## English

This is the public publication repository for the Swedish translation of William Shotts' _The Linux Command Line: A Complete Introduction, Third Edition_.

### Original work and permissions

- original work page: <https://linuxcommand.org/tlcl.php>
- original title: _The Linux Command Line: A Complete Introduction, Third Edition_
- author: William Shotts
- original work license: `CC BY-NC-ND 3.0 US`
- additional permission: William Shotts has granted permission for this project to publish the Swedish translation as generated HTML and PDF editions
- not publicly distributed: the private AsciiDoc translation source

### Free-work status

This publication is not a free cultural work or free documentation edition. The original work remains licensed under `CC BY-NC-ND 3.0 US`, and the Swedish translation is published under an additional permission limited to generated HTML and PDF editions.

Within those limits, the public editions are kept as open as practical:

- the HTML reading edition is static and ships without JavaScript
- the HTML edition uses local stylesheets, self-hosted libre webfonts, and no external runtime assets
- the PDF edition uses libre third-party fonts; see [`THIRD_PARTY.md`](THIRD_PARTY.md)

### Accessibility and standards

The public HTML reading edition is intended to be as accessible and standards-friendly as practical within a static, no-JavaScript publication model.

- target: aim for `WCAG 2.2 AA`-level good practice for the published HTML edition
- markup model: generated static HTML with semantic landmarks added in the export step
- interaction model: keyboard-usable, visible focus states, reduced-motion support, and no JavaScript dependency
- asset model: local stylesheets and self-hosted libre fonts only

This is an engineering target, not a formal certification. Conformance still depends on periodic manual testing with browsers, keyboards, zoom/reflow checks, and assistive technology.

### What this repository contains

- project documentation and publication notices
- generated HTML output for GitHub Pages
- generated PDF release artifacts
- selected build and publication tooling for transparency and reuse

The private AsciiDoc translation source is not distributed here. The `main` branch files in this public repo are maintained directly here, while generated HTML and the staged PDF come from the private source workspace.

### Related repositories and outputs

- public repo: `kommandoraden-i-linux`
- public Pages branch: published from the same repo
- PDF release asset: published from the same repo's releases
- private source repo: kept separate and not published

## Svenska

Detta är det publika publiceringsarkivet för den svenska översättningen av William Shotts _The Linux Command Line: A Complete Introduction, Third Edition_.

### Originalverk och tillstånd

- sida för originalverket: <https://linuxcommand.org/tlcl.php>
- originaltitel: _The Linux Command Line: A Complete Introduction, Third Edition_
- författare: William Shotts
- originalverkets licens: `CC BY-NC-ND 3.0 US`
- ytterligare tillstånd: William Shotts har gett tillstånd för att publicera den svenska översättningen som genererade HTML- och PDF-utgåvor
- publiceras inte offentligt: den privata AsciiDoc-källan för översättningen

### Status som fritt verk

Den här publiceringen är inte ett fritt kulturverk och inte en fri dokumentationsutgåva. Originalverket är fortfarande licensierat under `CC BY-NC-ND 3.0 US`, och den svenska översättningen publiceras med ett ytterligare tillstånd som är begränsat till genererade HTML- och PDF-utgåvor.

Inom de ramarna hålls de publika utgåvorna så öppna som det är praktiskt möjligt:

- HTML-utgåvan för läsning är statisk och levereras utan JavaScript
- HTML-utgåvan använder lokala stilmallar, självhostade fria webbtypsnitt och inga externa runtime-resurser
- PDF-utgåvan använder fria tredjepartstypsnitt; se [`THIRD_PARTY.md`](THIRD_PARTY.md)

### Tillgänglighet och standarder

Den publika HTML-utgåvan för läsning är tänkt att vara så tillgänglig och standardvänlig som det är praktiskt möjligt inom en statisk publiceringsmodell utan JavaScript.

- mål: sikta på god praxis i nivå med `WCAG 2.2 AA` för den publicerade HTML-utgåvan
- markupmodell: genererad statisk HTML med semantiska landmärken som läggs till i exportsteget
- interaktionsmodell: användbar med tangentbord, synliga fokusmarkeringar, stöd för minskad rörelse och inget beroende av JavaScript
- resursmodell: endast lokala stilmallar och självhostade fria typsnitt

Detta är ett tekniskt mål, inte en formell certifiering. Faktisk överensstämmelse beror fortfarande på återkommande manuell testning i webbläsare, med tangentbord, zoom/reflow-kontroller och hjälpmedel.
