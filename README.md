# Kommandoraden i Linux

Det här är publiceringsarkivet av den svenska översättningen av William Shotts _The Linux Command Line: A Complete Introduction, Third Edition_.

- HTML-format: https://kommandoraden-i-linux.l10n.se/
- PDF-format: https://github.com/itiquette/kommandoraden-i-linux/releases

Utgåvan är inte ett fritt verk — se [`PERMISSIONS.md`](PERMISSIONS.md) för originalverk, licens och publiceringstillstånd.
Inom de ramarna hålls utgåvorna så fria och öppna som det är praktiskt möjligt.

Utgåvorna är så tillgängliga och standardvänliga som det är praktiskt möjligt som exempelvis fria typsnitt, och god tillgänglighetspraxis i nivå med `WCAG 2.2 AA`.

## English

This is the publication repository of the Swedish translation of William Shotts' _The Linux Command Line: A Complete Introduction, Third Edition_.

- HTML edition: https://kommandoraden-i-linux.l10n.se/
- PDF edition: https://github.com/itiquette/kommandoraden-i-linux/releases

This is not a free work — see [`PERMISSIONS.md`](PERMISSIONS.md) for the original work, license, and publication permission.
Within those limits, the editions are kept as free and open as practical.

The editions are as accessible and standards-friendly as practical — for example libre fonts and accessibility practice at the level of `WCAG 2.2 AA`.
